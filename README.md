# APP-STORE-NODE

## Install

```shell
npm install
```

<span style="color:red">Important !!!</span> -> **You must copy .env.example to .env and adapt your configuration to the project**

```shell
cp .env.example .env
```

To initialize the database you can run:

```shell
npm run install-db
```

## Run the app

Start the api:

```shell
npm start
```
